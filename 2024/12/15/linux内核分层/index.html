Posted on 2024-12-15

Hardware

硬件底层，包括CPU、内存、输入输出设备等。

Hardware Abstraction

硬件抽象层，封装硬件细节，提供统一接口。

Kernel Core

核心功能，包括进程管理、内存管理、文件系统、网络协议等。

System Call Interface

系统调用接口，为用户程序提供访问内核服务的入口。

User Space

用户空间，运行应用程序和库。
